각 디렉토리는 ABAP 프로그램 단위로 구성되어 있으며, 
해당 프로그램에는 Screen, Include, Function Module, Dictionary 객체들이 포함되어 있습니다.

```

├── ZJSJ_MASTER_CRUD       # 등급산출 기준 정보 CRUD 프로그램
```
